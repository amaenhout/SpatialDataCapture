API 1: Map Hexagons

http://dev.spatialdatacapture.org:8824/data/:area/:category
Based on the user’s selection of the area (country/Europe) and category, an API obtained and filtered the table (‘sbs_hexcountries’) based on the selected category-of-interest, before returning the values. The hexagon IDs were then matched to the hexagon IDs of the relevant GeoJSON files that were loaded locally, so that each hexagon of the area-of-interest had a category value from the database.

Parameter Type # Example
Area string Required Europe / Austria
Category string Required Art_Culture 
[
    {"hex":"8418441ffffffff","value":7,"sum":30},
    {"hex":"8418443ffffffff","value":114,"sum":593},
    {"hex":"8418445ffffffff","value":49,"sum":258},
    ...
    {"hex":"8418447ffffffff","value":234,"sum":1117},
    {"hex":"8418449ffffffff","value":6,"sum":48},
    {"hex":"841844bffffffff","value":14,"sum":75},
]

API 2: Map Max Category

http://dev.spatialdatacapture.org:8824/map/:area
Based on the user’s selection of the area (country/Europe), our second API returned the hexagon IDs and the category that has the highest count for each hexagon. This API connected to (‘countriesmax’), a derivative of the main datatable, containing information about the category with the highest count for each hexagon.

Parameter Type # Example
Area string Required Europe / Austria 
[
    {"country":"austria","hex":"851e0253fffffff","max_value":26,"cat":"geography"},
    {"country":"austria","hex":"851e02c3fffffff","max_value":183,"cat":"geography"},
    {"country":"austria","hex":"851e02c7fffffff","max_value":57,"cat":"geography"},
    ...
    {"country":"austria","hex":"851e02cbfffffff","max_value":23,"cat":"geography"},
    {"country":"austria","hex":"851e02cffffffff","max_value":33,"cat":"geography"},
]

API 4: Period chart

http://dev.spatialdatacapture.org:8824/periodchart
The fourth API provided time-specific data aggregated by countries. The percentage of existing street names represented by each time period from 1600 to 2020 was calculated, and added cumulatively to return the percentage of streets before each time period.

Chart string Unrequired

API 5: Radar chart City

http://dev.spatialdatacapture.org:8824/radarchartcity/:city
The first part of the fifth API provided the data necessary for plotting the first city radar chart layer, aggregated by city where the city has the proportion of total streets in each category.

API 6: Top Streets for Hexes

http://dev.spatialdatacapture.org:8824/tophex/:hex/:country/:category
The last API returns the top streets within a hexagon on the map, this taken in account the country (to make sure the hex is in the country) and the category the user selected

Parameter Type # Example
Hex string Required 841fa6dffffffff
Country string Required Europe / Austria
Category string Required Art_Culture
